# 🏠 House Price Prediction

## 📌 Project Overview
This project predicts house prices based on area and number of bedrooms using Machine Learning.

## 🚀 Tech Used
- Python
- Pandas
- Scikit-learn

## 📊 Model
- Linear Regression
- Accuracy: 99%

## ▶️ How to Run
1. Install libraries
2. Run model.py

## 💡 Example
Input: area=1500, bedrooms=3  
Output: Predicted price
